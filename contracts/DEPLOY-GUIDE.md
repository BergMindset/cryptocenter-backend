# Деплой контракта подписки PoolsSubscription — пошагово (делает основатель)

> Claude контракт НЕ деплоит (ключи/газ/смарт-контракты — зона основателя, твёрдое правило).
> Эта инструкция — чтобы ты задеплоил сам за ~5 минут через Remix (браузер + твой кошелёк).
> ⚠️ Контракт неизменяем после деплоя и тянет $20/мес с реальных кошельков. Внешнего аудита нет
> (внутренний адверсариальный — прошёл, 6 must-fix внесены). Реши осознанно. После деплоя —
> ОБЯЗАТЕЛЬНО сам подпишись на $20 и отмени, ДО того как открывать доступ публике.

## 0. Что нужно приготовить (от тебя)
- **treasury** — адрес казны, куда идут все $20-платежи. ВЗЯТЬ СТРОГО ИЗ ЦЕНТРА (admin.bergapp.com),
  сверить побайтово (у нас была история с отравлением адресов). Это НЕ адрес из письма/чата.
- **owner** — твой кошелёк-владелец (может ставить цену/паузу/менять казну через таймлок). Лучше мультисиг.
- Кошелёк с небольшим количеством **ETH на Base** для газа (~$1–2).

## 1. Аргументы конструктора (готовы, менять только treasury/owner)
```
_token     = 0x833589fCD6eDb6E08f4c7C32D4f71b54bdA02913   // USDC на Base (6 decimals)
_treasury  = <АДРЕС КАЗНЫ ИЗ ЦЕНТРА>
_price     = 20000000                                       // $20 (20 × 10^6)
_maxPrice  = 50000000                                       // потолок $50 (иммутабельный)
_owner     = <ТВОЙ КОШЕЛЁК-ВЛАДЕЛЕЦ>
```

## 2. Деплой через Remix (remix.ethereum.org)
1. Открой https://remix.ethereum.org
2. Создай файл `PoolsSubscription.sol`, вставь ВЕСЬ код из нашего `contracts/PoolsSubscription.sol`.
3. Слева «Solidity Compiler» → версия **0.8.24** → «Compile». Зависимости OpenZeppelin Remix подтянет сам
   (import из `@openzeppelin/contracts` — Remix резолвит из npm автоматически).
4. Вкладка «Deploy & Run»:
   - Environment → **Injected Provider - MetaMask** (кошелёк должен быть на сети **Base**, chainId 8453).
   - Contract → `PoolsSubscription`.
   - Разверни поле рядом с «Deploy», впиши 5 аргументов из п.1 по порядку.
   - Нажми **Deploy**, подтверди транзакцию в кошельке.
5. Скопируй адрес задеплоенного контракта.

## 3. Верификация на Basescan (чтобы был публично проверяемым)
- Basescan → адрес контракта → Verify & Publish → Solidity (Single/Standard-JSON) → compiler 0.8.24 →
  вставь исходник/загрузи. Либо через Remix-плагин «Etherscan - Contract Verification».

## 4. Пришли Claude
- Адрес задеплоенного контракта.
- Подтверди, что treasury сверён с Центром.

## 5. Что дальше сделает Claude (после твоего деплоя)
- Пропишет `SUBSCRIPTION_CONTRACT=<адрес>` в env бэкенда DO + пересоберёт (гейт включится автоматически).
- Построит кнопку «Подписаться $20/мес» (permit-подпись) на /pools и протестирует её ПРОТИВ ТВОЕГО ЖИВОГО
  контракта на малой сумме, ДО публичного анонса.
- Отдельно (твоя зона): крон авто-продления `charge()` — одноразовый газовый ключ, по SIGNER-SPEC/SUBSCRIPTION-SPEC.

## Проверочный чек-лист перед публикой
- [ ] treasury сверён с Центром побайтово
- [ ] контракт верифицирован на Basescan
- [ ] ты сам подписался на $20 и увидел списание В КАЗНУ (не куда-то ещё)
- [ ] ты вызвал cancel() и убедился, что автопродление выключилось
- [ ] только после этого — открываем гейт публике
